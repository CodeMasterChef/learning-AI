Số nguyên: 1

Số thực: 1.0

Các phép toán như các ngôn ngữ khác: + , - , * , / , % (lấy số dư). 

Phép toán đặc biệt: Lũy thừa: `**`. 
```
Ví dụ: 2**4 = 16
```

Khai báo biến không cần kiểu dữ liệu. Ví dụ: 

```
myVar = 2
```

Kiểu dữ liệu thì có thể dùng '' hoặc "". Ví dụ: 

```
'single quote'
"double quote"
```
